# SkyFlow Optimized | 空流优化整合包

> 一个专为 Minecraft Fabric 打造的**轻量级优化整合包**，旨在彻底消除原版卡顿，提供流畅、稳定的生存体验。

---

## ✨ 整合包特色

- **🚀 极致流畅** - 搭载 Sodium、Lithium 等核心优化模组，帧数飙升，告别卡顿。
- **🎨 视觉升级** - 支持 Iris 光影，并集成 ETF、EMF 等纹理模型优化，画面更细腻生动。
- **⚙️ 体验拉满** - 包含 Jade（玉）、一键背包整理、边拿边走等必备功能，便捷性全方位提升。
- **🔊 沉浸音效** - 新增环境音效与物理声学，营造更具临场感的冒险氛围。
- **🧩 开箱即用** - 模组经过精心配置与兼容性测试，下载即玩，无需繁琐调整。

## 📥 下载与安装

### 最新版本下载
请前往 **[Release](../../releases/)**或者**[modrinth](https://modrinth.com/modpack/skyflow-optimized)** 页面下载最新的 `.mrpack` 整合包文件。

### 快速启动指南
1.  **下载** 最新的 `SkyFlow-Optimized-x.x.x.mrpack/空流整合包 x.x.x.mrpack` 文件。
2.  **拖拽** 该文件到您的 PCL2/HMCL（未测试） 等支持 `.mrpack` 的启动器窗口。
3.  **启动** 启动器会自动创建实例并安装所需文件，点击“启动”即可畅玩！

## 🛠️ 系统要求

- **内存**：建议分配 **6GB** 以上内存。
- **光影**：若使用光影，建议分配 **8GB+** 内存。
- **存储**：整合包本体约占 **200MB** 空间。
